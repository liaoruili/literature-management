- **任务队列**: Celery

- **AI**: OpenAI API / Claude API

- **包管理**: UV (Python) / npm (Node)

- **测试**: pytest / Vitest

## 📝 开发规范

### Git 工作流
- 使用 Feature Branch 工作流
- 分支命名: `feature/功能名`, `fix/问题描述`
- Commit 规范: `type(scope): description`
- 主要类型: feat, fix, docs, style, refactor, test, chore

### 代码规范
- Python: PEP 8, 使用 Black 格式化
- TypeScript: ESLint + Prettier
- 函数长度不超过 50 行
- 类长度不超过 300 行
- 注释率不低于 20%

### 测试要求
- 核心 API 覆盖率 > 80%
- 关键 UI 组件有单元测试
- 集成测试覆盖主要用户流程
